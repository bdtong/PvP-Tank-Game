IDE: NetBeans

Java Version: JDK 1.8

How to run:
	1) Create git repo.
	2) git clone https://github.com/csc413-03-sp18/csc413-03-tankgame-Team15-bdtong.git
	3) Set current working directory to "TankGame" folder.
	4) Run.

Game Controls:
	Player 1 - Movement "WSAD"
		 - Fire "Space Bar"
	Player 2 - Movement "Up, Down, Left, Right Arrow Keys"
		 - Fire "Numpad 0"
